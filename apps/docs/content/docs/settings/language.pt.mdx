# Idioma

No menu de configurações da CLI você pode selecionar a opção `Idioma` e alterar o idioma padrão da CLI, isso se aplica apenas aos textos exibidos na CLI, as logs e o README do projeto gerado ainda estarão no idioma inglês (seguindo boas práticas de programação).

Abaixo você confere os idiomas disponívies:

- Inglês
- Português
