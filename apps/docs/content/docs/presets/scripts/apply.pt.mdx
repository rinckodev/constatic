# Aplicando predefinições de scripts

Você pode aplicar predefinições existentes a um projeto já criado usando essa opção

Depois de [acessar o menu de predefinições de scripts](/docs/presets/scripts/start) você pode selecionar a opção para aplicar:

<CliMenus.Scripts lang="pt" selected="apply"/>

Você poderá selecionar todas as predefinições que quiser e os arquivos serão copiados para o diretório em que a CLI está sendo executada da [forma que foi estruturada na criação](/docs/presets/scripts/new)
