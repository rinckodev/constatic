# Editando predefinições de scripts

Você pode editar uma predefinição criada anteriormente usando essa opção

Depois de [acessar o menu de predefinições de scripts](/docs/presets/scripts/start) você pode selecionar a opção para editar:

<CliMenus.Scripts lang="pt" selected="edit"/>

Então escolha uma predefinição para editar e você verá o mesmo menu de criação de predefinição. Faça as alterações que precisar e clique em `Salvar`.
