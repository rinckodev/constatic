As predefinições de scripts são uma forma de reutilizar código de forma fácil e simples com a CLI. Você pode agrupar arquivos de código e instruções em uma predefinição e reutilizar ao gerar um novo projeto ou gerar em projetos já existentes. Dessa forma o trabalho manual de copiar e colar arquivos é reduzido.

# Começando

<Steps>
    <Step>
        Para usar este recurso comece executando a CLI:

        <CliCommand packageName="constatic@latest" />
    </Step>
    <Step>
        Então no menu principal selecione a opção `Gerenciar predefinições`.
        <CliMenus.Main lang="pt" selected="presets"/>
    </Step>
    <Step>
        Depois disse selecione as predefinições de scripts:

        <CliMenus.Presets lang="pt" selected="scripts"/>
    </Step>
</Steps>

Depois disso você verá o menu onde poderá criar e gerenciar predefinições de scripts
