# Creating a Firestore Database
You need to have a Firebase account!

If you don't have one, please go to https://console.firebase.google.com/ and log in with your Google account.

Let's create a project

<ImageZoom src={"/firestore/step-01.png"} width={500} height={500} alt="firestore-step-01" />

Give your project a name and continue

<ImageZoom src={"/firestore/step-02.png"} width={500} height={500} alt="firestore-step-02" />

Leave this option unchecked if you prefer

<ImageZoom src={"/firestore/step-03.png"} width={500} height={500} alt="firestore-step-03" />

Now click on Firestore Database in the sidebar

<ImageZoom src={"/firestore/step-04.png"} width={500} height={500} alt="firestore-step-04" />

Create a new database

<ImageZoom src={"/firestore/step-05.png"} width={500} height={500} alt="firestore-step-05" />

Leave "production mode" checked and click next

<ImageZoom src={"/firestore/step-06.png"} width={500} height={500} alt="firestore-step-06" />

Choose the location of your database (recommended to be closer to your application's location)

<ImageZoom src={"/firestore/step-07.png"} width={500} height={500} alt="firestore-step-07" />

Now you have a Firestore database

<ImageZoom src={"/firestore/step-08.png"} width={500} height={500} alt="firestore-step-08" />

Access the project settings

<ImageZoom src={"/firestore/step-09.png"} width={500} height={500} alt="firestore-step-09" />

Go to service accounts and click on generate new private key

<ImageZoom src={"/firestore/step-10.png"} width={500} height={500} alt="firestore-step-10" />

You will be able to save this file on your computer

<ImageZoom src={"/firestore/step-11.png"} width={500} height={500} alt="firestore-step-11" />

Rename the file to "firebase.json"

<ImageZoom src={"/firestore/step-12.png"} width={500} height={500} alt="firestore-step-13" />

Save it in the root of your project

You're ready to use it!
