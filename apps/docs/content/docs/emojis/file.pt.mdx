# Arquivo de emojis

Com a opção `Gerar arquivo de emojis` você pode obter um arquivo json com todos os nomes e IDs dos emojis da sua aplicação para que você possa utilizar nos textos e componentes do seu projeto de bot através de funções utilitárias.

Você poderá informar o nome e caminho do arquivo (Por padrão será `emojis.json`). Mas você pode escolher outro nome para o arquivo como `emojis.dev.json` ou até mesmo passar o caminho para alguma pasta `./assets/emojis.json`.

Logo em seguida você poderá escolher o modo que os emojis serão listados no arquivo
- IDs - Os valores de todas as propriedades são apenas os IDs dos emojis
- URLs - Os valores de todas as propriedades são as URLs cdn dos emojis
